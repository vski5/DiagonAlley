## Foundry

**Foundry 是一个用 Rust 编写的针对以太坊应用开发的快速、便携和模块化工具包。**

Foundry 包括：

-   **Forge**：以太坊测试框架（类似于 Truffle、Hardhat 和 DappTools）。
-   **Cast**：用于与 EVM 智能合约交互、发送交易和获取链上数据的多功能工具。
-   **Anvil**：本地以太坊节点，类似于 Ganache、Hardhat Network。
-   **Chisel**：快速、实用且详细的 Solidity REPL。
